Python version : 3.11.3
Pygame version : 2.1.2

All games parameters are set in utilitaries.py

When DEV_MODE = True, list of shortcuts:

G : Game Over
N : gold x2
K : Clear all ennemies
H : Set all ennemies HP at 0

0-9 : Ennemy spawning (max 1 in the same time)

To stop natural spawnin : TURN_OFF_NATURAL_SPAWNING = True
